# Streamlit Calculator App

An interactive calculator web application built with Streamlit, enabling users to perform real-time arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Input validation for incorrect values

## Tech Stack
- Python
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
